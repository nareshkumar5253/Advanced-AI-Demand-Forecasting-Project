#  Advanced AI Demand Forecasting System

An end-to-end Machine Learning project that predicts product demand using historical data and provides an interactive web dashboard for visualization and insights.

---

##  Overview

This project uses Machine Learning techniques to forecast demand and help businesses optimize inventory, reduce waste, and improve decision-making.

It includes:
- ML model for demand prediction
- Backend API for inference
- React frontend dashboard
- Data upload and visualization features

---

## Tech Stack

**Frontend:**
- React.js
- Vite
- Tailwind CSS

**Backend:**
- Python
- Flask / FastAPI
- Pandas, NumPy
- Scikit-learn
